# salted-fish

> i am not a salted fish

## Build Setup

``` bash
# 安装依赖
npm install

# 启动项目
npm run dev

# 打包文件
npm run build

```

## git 提交
```
- 拷贝项目：git clone 'github地址' 
- 添加文件：git add .
- 提交信息：git commit -m '注释'
- 上传更新：git push -u origin master
```
